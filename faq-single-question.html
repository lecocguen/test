FAQ / Nom de domaineComment transferer un nom de domaine vers Sportsregions ?

Si vous êtes déjà propriétaire d’un nom de domaine et que vous souhaitez que votre site Sportsregions soit accessible à travers celui , vous devez effectuer une procédure de transfert.
Pour cela, prenez contact avec le registrar de votre nom de domaine actuel afin de dévérouiller le nom de domaine. Récupérez le code d'authentification / clé de transfert pour ensuite nous le transmettre afin que nous puissions lancer le transfert.
Chez certains registars cette opération peut se faire directement depuis l’espace d’administration.
Si votre nom de domaine est déjà chez OVH il n'est pas nécessaire de le transférer. Nous vous communiquerons notre identifiant OVH afin que vous puissiez l'enregistrer comme Administrateur de votre nom de domaine.
